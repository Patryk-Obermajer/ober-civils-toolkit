Ober Civils Toolkit User Manual

Welcome to the manual for Ober Civils Toolkit, the cutting edge 2D ground modeling and drainage design pack.
Ober Civils Toolkit is the result of engineers wanting a better way to create, review and update their design information. Designed by engineers specifically for the most intuitive and fastest possible worflow when working with tight deadlines.



Key features

  • Ober Civils Toolkit is a 2D ground modelling package, offering a broad range of tools enabling seamless work with engineering levels including, but not limited to, reviewing and calculating distance, level difference and gradient, quickly creating new level information based on either level difference or gradient, intutively generating spot levels around the proposed plots, and batch raising or lowering of spot levels. Similarly, it enables simplified creation of lateral drains in 2D.
  • It offers speed and reliability largely thanks to being 2D based. Computing power is not being wasted on recalculating and re-triangulating of the 3D information. The toolkit enables us, however, to covert the generated 2D levels to 3D to be then exported and processed.
  • It contains many features, which help simplify and automate common oftentimes repetitative tasks, e.g. setting out, working with output drawings from PDS, managing layers, and many others.

Configuration

Toolkit comprises two componentst: Ribbon user interface (Ober Civils Toolkit.cuix) and back-end program (Ober Civils Toolkit.fas):

  • To load the Ribbon TAB onto your AutoCAD, type CUILOAD into the command bar. Go to the \Ober Civils Toolkit folder and LOAD Ober Civils Toolkit.cuix. This has to be performed only once and the ribbon tab will open every time AutoCAD launches.
  • To load the FAS executable file, type APPLOAD. Go to the \Ober Civils Toolkit folder and load Ober Civils Toolkit.fas. Then, still in Load/Unload application window, navigate to CONTENTS within Startup Suite tab (bottom right corner) and add Ober Civils Toolkit.fas to ensure the Toolkit loads every time AutoCAD is opened.


NOTE: Toolkit only works with full version of AutoCAD

Levels

Gradient/Fall Calculator

Measures distance between two points. Prompts user for either fall or gradient. Calculates gradient and fall between the two points.

Spot Level Calculator

Measures distance between two spot level blocks. Calculates the level difference and gradient between the points. Optionally adds a label in-between.

Spot Level/FFL Creator

User selects the base Spot Level or FFL block. Prompts for new block location, level difference or gradient. Creates a new spot level/FFL block in the selected location.

Spot Levels Around FFL

Generates Spot Levels around FFL. By default spot levels around FFL are 150 mm below the FFL unless Underbuild option has been selected, in which case the spot level is reduced by 150 mm + Underbuild Height (example: UB height = 300 mm. New spot level at reduced level by 450 mm).

Raise/Lower Levels

Adjusts both FFL and Spot Levels by entered value. Enables also to batch adjust levels of a number of blocks.

Move Block To Elevation

Converts 2D spot levels to 3D: by moving the blocks to elevation based on block attribute value.

Text To Block

Inserts blocks based on Text Value. Only works with “Text” and not “MTEXT”. “MTEXT” can be exploded in order to convert to “TEXT”

Lateral Drains

  • SW PIPE DIA and FW PIPE DIA – Sets pipe diameter of Lateral pipe
  • SW LATERALS and FW LATERALS – Prompts for upstream chamber Invert Level label (only “TEXT” entities), then centre of upstream and downstream chambers and downstream Invert Level Label.

Layer Operators

  • Set Up PDS – Sets PDS Layers within the Drawing to correct colours and converts “CLAY” material text to “uPVC”
  • Set Up Engineering – Batch changes colours within Xrefs (only works when xref naming convention is followed: x-layout, x-survey, etc.) SW LATERALS and FW LATERALS – Prompts for upstream chamber Invert Level label (only “TEXT” entities), then centre of upstream and downstream chambers and downstream Invert Level Label.

Miscellaneous

Setting out

Sets outs coordinates of each vertex of selected polylines and lines. Sets out either as Mleaders or Blocks (prompts user to select the setting out block – see Blocks.dwg). Block Setting Out option works only with blocks, which have only one attribute.

Circles to Blocks

Prompts user to select the base block and then circles. Inserts blocks in the centre points of the selected blocks. Works only with non-dynamic blocks (which have no attributes).

2D-Distance

Lets measure 2D distance even when Snap is On and objects are on different elevations.

Select Previous

Selects previously selected objects.

Title Block Attribute Transfer

Enables attribute transfer across attributed blocks. Especially useful when applied to Title Blocks of different Sheet Sizes (say A1 to A2 etc.)

OS Grid Reference Finder

Prompts for a point. Grid Reference Finder website pops-up
